Installing Roadrunner

As first step, please download the vert.x based Roadrunner distribution standalone download. You can start it, by simply executing the follwing command 

java -jar roadrunner-vertx-0.1-standalone.jar
This will start a standalone vert.x based Roadrunner on port 8080 of your local computer. Now you can start using Roadrunner in your web app for development.

To get started add the Roadrunner JavaScript client by adding the following line into the section below: 

<script src='http://localhost:8080/roadrunner.js'></script>

Accessing Data

Roadunner references are created using a URL that specifies which data you want to access.
To complete this step, create a reference to the root of your Roadrunner repository as shown below:

var myDataRef = new Roadrunner('http://localhost:8080/');

Writing Data

Let's send a chat message You can use the Roadrunner reference you just created to write data to Roadrunner using the set() function.
To make things easy, we've already added input text boxes for the chatter's name and message as well as a keypress handler that will fire whenever someone tries to send a message.
For this step, write a message to Roadrunner using the set() function as shown:

myDataRef.set('User ' + name + ' says ' + text);

Writing Objects

The set() function can also be used to write objects.
Try changing your code to write an object with text and name properties:

myDataRef.set({name: name, text: text});

Writing Lists

You've already learned how to write data to specific, named locations in Roadrunner, but your chat application will require a list of messages. Roadrunner provides a helper function called push() that makes creating lists easy.
Modify your code below to use push() instead of set() so that your chat can support a list of messages (rather than just one):

myDataRef.push({name: name, text: text});

Reading Data

We need to tell Roadrunner to notify us when chat messages arrive. We do this by adding a callback to the list of chat messages using the on() method, as shown below:

myDataRef.on('child_added', function(snapshot) {
  //We'll fill this in later.
});
This method takes two arguments: the event type and the callback function. We'll use the 'child_added' event so that we are notified of the arrival of individual messages.

Using Snapshots

Now we need to display the chat messages on the page. For each chat message, Roadrunner will call your callback with a snapshot containing the message's data.
Extract the message data from the snapshot by calling the val() function and assign it to a variable. Then, call the displayChatMessage() function to display the message as shown:

var message = snapshot.val();
displayChatMessage(message.name, message.text);
